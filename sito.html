Flagship graphics cards comparison

Graphics cards work based on parallel calculations so more cores are better than higher speed.
Radeon VII uses 16GB next gen HBM2 memory instead of 8GB classic GGDR6 like the RTX 2080.
Radeon VII uses a next gen 7nm manifacturing process while RTX 2080 uses a classic 12nm manufacturing process.

Flagship processors comparison

Processors use cache to store instructions, split tasks between cores and run as fast as clocks go.
Ryzen 3950X has +450% bigger cache and double the cores of i9 9900K but slightly less clock speed.
Ryzen 3950X uses a next gen 7nm manifacturing process while 9900K uses a classic 14nm manufacturing process.
